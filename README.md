**Description:**

This project simulates the 3-dimensional motion of a quadcopter under various flight conditions, integrating rigid-body dynamics principles with numerical solutions to 12-state equations of motion. The simulation calculates the quadcopter's center of mass, moments of inertia, and predicts trajectories for vertical, horizontal, angled, and helical flight paths. The work was completed as part of a team.

**Files":**
ThreeDQuadRunner1.mlx - Main script; calculates mass properties and runs all 3D simulations
ThreeDQuad11.m - Function for solving the 12-state equations of motion for vertical and horizontal flights
ThreeDQuad21.m FUnction for solving the 12-state equations of motion for helical flight

**Usage:** 
1. Clone the repository: git clone <repository-url>
2. Open MATLAB and add the repository folder to your path
3. Run the main simulation file (ThreeDQuadRunnner1.mlx)
4. Observe trajectory plots and velocity profiles generated for each scenario
